# Partner-informed listing taxonomy (Jeffreys Bay pilot)

Co-created listing shapes with anchor NGOs/churches — evolve via workshops; treat this file as **living guidance**, not legal advice.

## Cross-cutting fields (all tabs)

| Field | Notes |
|-------|-------|
| `organisation` | Listing steward (e.g., Joshua Project, Wave Point Foundation). |
| `title` | Plain-language offer title. |
| `summary` | Short description for cards. |
| `schedule` | Cadence or booking notes (structured later). |
| `proof_requirements` | Which credentials/skills evidence residents must show privately. |

## Education tab exemplars

- Adult Education & Training (AET) modules (Joshua Project).
- Life skills / readiness curricula tied to Work 4aLiving-style pathways.
- Entrepreneurship / leadership labs (Victory Gap Year collaborations).

## Recreation tab exemplars

- Surf mentorship / leadership sessions (Wave Point Foundation, Christian Surfers pipelines).
- Agricultural or gardens volunteering slots referencing Wave Point Gardens.

## Entertainment tab exemplars

- Youth cultural events, ticketing coordination with privacy-preserving attendance proofs (Victory Frontline Youth collaborations).

## Moderation norms

- Trusted organisations seed initial listings; community governance (treasury votes) defines escalation paths for disputes — capture specifics in workshop notes under `docs/partners/workshops/YYYY-MM-DD.md` as they occur.

## Broader faith network

Jeffreys Bay Bible Church, Methodist, Catholic, Adventist, and similar congregations focus on **education/comms** channels rather than bespoke schema fields unless programmes demand unique metadata.
