## Privacy Briefing (read to participant before collecting consent)

### What this pilot collects

- **Profile data you choose to share:** Name, role, service description, contact method (email/phone — entirely optional for browsing)
- **Usage telemetry:** Screen views, feature taps, time-on-app — anonymous, no personal identifier attached
- **Identity verification (optional):** If you opt in to the verified badge, a zero-knowledge proof is generated. The app never stores your ID number — only a cryptographic proof that you are a real person.

### What this pilot does NOT collect

- ID numbers, home addresses, financial data
- Location tracking (GPS is never queried)
- Social media accounts or passwords
- Communication content (messages, reviews)

### Your rights (POPIA)

1. **Access** — view all data we hold about you via app settings
2. **Correction** — update your profile anytime
3. **Deletion** — delete your account and all associated data with one tap
4. **Withdrawal** — revoke consent at any time without penalty
5. **Complaint** — contact the CSALT coordinator or info@jbayhub.xyz if you have concerns

### Data handling

- Consent is stored as a **verifiable credential** on Midnight testnet — you hold a copy
- Profile data lives in encrypted storage
- Telemetry is aggregated and anonymized before analysis
- Retention: data kept for 12 months post-pilot unless you request earlier deletion
