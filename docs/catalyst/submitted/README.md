# Submitted Catalyst Artifacts

This directory stores evidence after the Fund 15 proposal is submitted.

## Contents

| File | Purpose |
|------|---------|
| `statement-of-milestones-final.md` | Anonymized copy of the submitted SoM (no secrets) |
| `proposal-link.txt` | Canonical Catalyst proposal URL |
| `submission-date.txt` | Date of submission |

## Instructions

1. After submitting in Catalyst Keychain, update `proposal-link.txt` with the
   canonical proposal URL.
2. Paste the final SoM text into `statement-of-milestones-final.md` (omit wallet
   addresses, recovery phrases, and any secrets).
3. Record the submission date in `submission-date.txt`.
4. Update the umbrella issue INK-88 comment with links to these files.

See: `docs/catalyst/catalyst-fund15-requirements.md` (prerequisites)
See: `docs/catalyst/statement-of-milestones-template.md` (SoM template)
